### What is TypeScript?
* superset of JS
* all JS is valid TS
* SCSS is a superset of CSS
* all CSS is valid SCSS

* static types
* eliminate most runtime errors

* .scss => node-sass => .css
* .ts => typescript compiler (tsc) => .js
